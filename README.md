# FUTURE__ML_01
# Task 1 - Sales & Demand Forecasting
## Future Interns - Machine Learning Internship

## Overview
Built a Machine Learning model to forecast future 
sales using historical business data.

## Tools Used
- Python
- Pandas
- Scikit-learn (Random Forest)
- Matplotlib & Seaborn

## What It Does
- Cleans and processes sales data
- Extracts time-based features
- Trains Random Forest model
- Predicts next 6 months of sales
- Generates business insights

## Files
- Task1_Sales_Forecasting.ipynb → Main code
- sales_data.csv → Dataset
- monthly_sales_trend.png → Sales trend chart
- sales_by_category.png → Category chart
- actual_vs_predicted.png → Model accuracy chart
- future_forecast.png → Future predictions

## Results
- Forecasted 6 months of future sales
- Identified peak and low sales periods
- Generated business recommendations
